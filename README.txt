To run smallsh.c
    1. Compile with gcc --std=gnu99 -o smallsh smallsh.c 
    2. Run program as ./smallsh 
    3. Program will run in the terminal
    4. Junk file must exist in the same directory
    5. Input exit to quit the program
    6. To run p3testscript: chmod +x ./p3testscript then ./p3testscript > mytestresults 2>&1 
    7. Results in mytestresults file
